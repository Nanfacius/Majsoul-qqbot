使用腾讯官方接口简单实现的qqbot

数据爬取自雀魂牌谱屋

**使用指南**

	git clone https://github.com/Nanfacius/Majsoul-qqbot.git
	cd Majsoul-qqbot/
	npm i
	cp config-default.js config.js

打开config.js，输入AppID和AppSecret，如果没有的话去官网申请，https://q.qq.com/#/

	node app.js

**指令：**

    雀魂查询/雀魂信息 昵称

    雀魂查询/雀魂信息 金/金之间/玉/玉之间/王座/王座之间 昵称
